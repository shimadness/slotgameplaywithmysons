# TRIPLE SLOT

息子たちと遊ぶために作った、ブラウザ用スロットゲーム。
ゲームの遊び方や演出は、ゲームセンターのメダルゲームの挙動を参考にさせてもらっていますが、
グラフィック・効果音・配当・ロジックはすべて独自実装で、外部の画像/音源・著作物は一切使っていません。
Vite + TypeScript 製で、サーバー不要・ローカルだけで動きます。

「**パズル × スロット**」の連鎖ゲーム（3×3 DROP）と、王道の **5リール** スロットを
モード切替で遊べます。

---

## クイックスタート

```bash
npm install     # 初回のみ
npm run dev     # http://localhost:5173 が開く
```

### 配布用ビルド（ファイルだけで動く）

```bash
npm run build     # dist/ に出力（型チェック込み）
npm run preview   # ビルド結果をローカル確認
```

`vite.config.ts` で `base: "./"` にしてあるので、`dist/index.html` を
ダブルクリックして `file://` で直接開いても動きます。

## ゲーム内容

ヘッダーの **モード切替** で2種類を切り替えられます。

### ① 3×3 DROP（連鎖パズル × スロット）

- **3×3グリッド**。同じシンボルが **3つ以上つながると配当**（クラスター判定）。
- つながり方は **「道（通路）」** で決まる：**タテ・ヨコは常に接続、ナナメは中央マスに
  つながる場合のみ**。揃うと、つながった道がシンボル色に**点灯**して見える化される。
- 揃ったシンボルは消え、上から新しいシンボルが**ぷよぷよ風に落下**（重力加速＋着地の潰れ）
  → 再判定で **連鎖**。連鎖が伸びるほど **倍率UP**。
- **NEXT プレビュー管**：各列の上に「次に落ちてくる」シンボルを先読み表示。管からそのまま
  盤面へ滑り込む連続演出。
- **リール回転スピンイン**：出目はいきなり出ず、各列が回転してから左→右に停止
  （速度連動モーションブラー＋停止音）。
- **ワイルドファイブ ✨**：任意のシンボルを代用するワイルド。**消えずに最大5回使える**
  （残り回数を表示。使うたび -1 して落下、5回使い切ると消滅）。
- **連鎖を多く重ねると RUSH 突入**（フリープレイ＋配当倍率、連鎖でさらに上乗せ）。

### ② 5リール

- **5リール × 3段 / 10ライン**。左から連続で揃うと配当。
- **スキャッター 🌟 3個以上で RUSH（フリースピン）** 突入。個数で回数・倍率が変化。
- **リーチ演出**：4本目停止時に「2スキャッター」or「高配当4連」を検出すると、
  最終リールを引き伸ばして点滅＋緊張音で煽る。

### プレイヤー（3人別々のクレジット）

- 起動時に **「だれが あそぶ？」** でプレイヤーを選択（3人分）。名前はその場で変更可。
- 各プレイヤーは **クレジット・ベット・設定が完全に別々**（端末内に個別保存）。
- ヘッダーの **👤 ボタン** でいつでも切り替え。各自のスマホで自分の名前を選べばOK。
- ※サーバーを使わない構成のため、クレジットは**端末ごとに保存**（端末間では同期しません）。

### 共通

- **効果音**：Web Audio ですべて合成（音源ファイル不要）。🔊 で消音切替。
- **RUSH 専用 BGM ＆ 背景演出**：RUSH中はループBGMが鳴り、背景が熱色に変化＋放射状の光が回転。
- **オートプレイ**：AUTO で自動連続プレイ。残高不足で自動停止。
- **クレジット/ベット**：BET▲ でベット変更、MAX BET、残高は localStorage に自動保存。
  残高不足で「+1000」補充ボタンが出ます。

### ペイアウト率（設定 1〜6）

ゲーセンの「設定」のように、**ペイアウト率（RTP）を設計値として持つ**仕組み。

- ヘッダーの **「設定」** ボタンで **1〜6** を切替（90% / 92% / 94% / 95% / 96% / 97%、既定は4＝95%）。
- 配当は内部で `目標RTP ÷ 基準RTP` の係数を掛けて、狙ったRTPに合わせ込んでいる。
- RTPは計測ハーネスで実測できる：

```bash
npm run rtp          # 既定スケールで計測
npm run rtp 0.146    # スケールを指定して計測
```

## 操作

| 操作 | 内容 |
| --- | --- |
| SPIN / Space / Enter | スピン |
| 👤 プレイヤー | プレイヤー（3人）を切り替え |
| BET ▲ | ラインベット 1→2→3→5→10 を循環 |
| MAX BET | 最大ベット |
| 設定 | ペイアウト率 1〜6 を切替 |
| AUTO | オートプレイ |
| 🔊 | 消音切替 |
| 配当表 | 遊び方・配当一覧を表示 |

## 構成

```
src/
  main.ts            進行管理・モード切替・設定（RTP）
  style.css          夜空キャビネットのスタイル
  game/
    symbols.ts       シンボル定義・配当・出現重み
    drop.ts          3×3 連鎖エンジン（クラスター判定／道／カスケード／ワイルドファイブ／RTP）
    paylines.ts      5リール10ライン定義と当たり判定
    engine.ts        リール帯生成・抽選
    state.ts         プレイヤー(3人)別 クレジット/ベット/RUSH/設定・セーブ
  audio/sfx.ts       Web Audio 効果音エンジン
  ui/
    dropBoard.ts     3×3 盤面描画＆連鎖アニメ（落下／道／プレビュー）
    board.ts         5リール描画＆回転アニメ
    effects.ts       ハイライト/勝利・連鎖ポップ/RUSHバナー/パーティクル
    hud.ts           メーターと操作ボタン
rtp.mjs              RTP（ペイアウト率）計測ハーネス
SPEC.md              実装仕様書（詳細）
```

詳しい仕様・実装の解説は [SPEC.md](SPEC.md) を参照。
